# מדריך תרומה לפרויקט

ברוכים הבאים לפרויקט האתר של דיאנה רחמני! אנו מעריכים את העזרה שלכם.

## תוויות (Labels) לניהול הפרויקט

אנו משתמשים בתוויות הבאות לניהול המשימות והבאגים ב-GitHub:

- **חֶרֶק (bug):** משהו לא עובד כראוי.
- **תיעוד (documentation):** שיפורים או תוספות לתיעוד הקיים.
- **לְשַׁכְפֵּל (duplicate):** בעיה זו או בקשת משיכה זו כבר קיימת.
- **הַגבָּרָה (enhancement):** תכונה או בקשה חדשה לשיפור.
- **גיליון ראשון טוב (good first issue):** מתאים במיוחד למפתחים חדשים בפרויקט.
- **דרוש/ה עזרה (help wanted):** נדרשת תשומת לב נוספת ממפתחים מנוסים.
- **לֹא בְּתוֹקֶף (invalid):** זה לא נראה נכון או לא רלוונטי.
- **שְׁאֵלָה (question):** מתבקש מידע נוסף או הבהרה.
- **וונטפיקס (wontfix):** החלטנו לא לעבוד על זה כרגע.

## תהליך העבודה

1. פתחו Issue חדש עם התווית המתאימה.
2. צרו Branch מהענף הראשי (`main`).
3. בצעו את השינויים והוסיפו בדיקות במידת הצורך.
4. פתחו Pull Request והמתינו לסקירה.
