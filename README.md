# אתר דיאנה רחמני - InsForge

אתר Next.js עבור דיאנה רחמני, עם InsForge כ-Backend.

## התקנה

```bash
npm install
```

## הרצה מקומית

```bash
npm run dev
```

פתח [http://localhost:3000](http://localhost:3000)

## פריסה ל-Vercel

1. העלה את הפרויקט ל-GitHub
2. התחבר ל-[vercel.com](https://vercel.com)
3. ייבא את הפרויקט
4. הוסף משתני סביבה:
   - `NEXT_PUBLIC_INSFORGE_URL`
   - `NEXT_PUBLIC_INSFORGE_ANON_KEY`
5. פרוס

## הגדרת InsForge

ראה [INSFORGE_SETUP.md](./INSFORGE_SETUP.md) להנחיות יצירת הטבלאות ו-Bucket.
