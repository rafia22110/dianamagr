# אתר דיאנה רחמני - InsForge

אתר Next.js רשמי עבור דיאנה רחמני, כוכבת מאסטר שף ומחברת "הבריחה מטהרן". האתר משלב מטבח פרסי, סיפורי השראה ומנגנונים עסקיים מתקדמים.

## תכונות מרכזיות

- **ניהול תוכן (CMS):** לוח בקרה למנהל להעלאת תמונות וניהול גלריה.
- **מערכת סדנאות והרצאות:** תצוגת אירועים קרובים ואפשרות הרשמה למשתמשים.
- **רכישת ספרים:** תהליך רכישה ישיר ומאובטח באתר.
- **ניהול רשימת תפוצה:** הרשמה לניוזלטר וניהול נרשמים.
- **אזור מנהל מאובטח:** ניהול מלא של כל הנתונים (הרשמות, הזמנות, נרשמים) דרך ממשק אחד.

## טכנולוגיות

- **Frontend:** Next.js 15, TypeScript, Tailwind CSS.
- **Backend:** [InsForge SDK](https://insforge.app) (Database, Auth, Storage).
- **Deployment:** Vercel.

## התקנה

```bash
npm install
```

## הרצה מקומית

1. העתק את `.env.example` ל-`.env.local` ומלא את הפרטים.
2. הרץ את שרת הפיתוח:
```bash
npm run dev
```

פתח [http://localhost:3000](http://localhost:3000)

## פריסה ל-Vercel

1. העלה את הפרויקט ל-GitHub.
2. התחבר ל-[vercel.com](https://vercel.com).
3. ייבא את הפרויקט והוסף את משתני הסביבה הנדרשים.

## הגדרת InsForge
